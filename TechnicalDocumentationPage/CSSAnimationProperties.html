CSS Animation Properties

Properties Basics Iteration count Direction Timing function

Properties

Using the animation properties and the @keyframes rule, elements can be animated using CSS. There are 8 animation properties:

  • animation-name is used to specify the name of the animation
  • animation-duration is used to specify how much time is seconds (s) or milliseconds (ms) the animation takes to complete one cycle
  • animation-timing-function is used to specify the speed curve of the animation; for example ease-in (slow start) or ease-out (slow end)
  • animation-delay is used to set a delay before the animation starts
  • animation-iteration-count is used to specify how many times an animation plays, which can be set to a number, or infinite
  • animation-direction is used to specify if an animation is played as normal, backwards, or alternating
  • animation-fill-mode is used to specify the style of the element when the animation is not playing
  • animation-play-state is used to specify if an animation is paused or not
The @keyframes rule is used call up the animation by it's animation-name property, and to declare which css-styles should be attributed to which points during the animation.

Basics

The minimum requirements for a CSS animation are the animation-name property, the animation-duration property, and the @keyframes rule with at least one CSS style declaration set to a point during the animation:

.example1 {
  height: 40px;
  width: 40px;
  background: red;
  animation-name: ChangeToOrange;
  animation-duration: 10s;
}
@keyframes ChangeToOrange {
  100% {background: orange;}
}

This code makes a simple red square, and then changes the background colour to orange over a duration of 10 seconds. After this it resets back to the starting color:

Iteration count

The animation-iteration-count property is used to make an animation repeat, either a set amount of times, or infinitely. The default value is 1, meaning the animation plays once and then resets to and stays in its initial state.

.example2 {
  height: 40px;
  width: 40px;
  background: blue;
  animation-name: ChangeToGreen;
  animation-duration: 5s;
  animation-iteration-count: infinite;
}
@keyframes ChangeToGreen {
  100% {background: green;}
}

This code takes a simple blue square, changes the background color to green over a duration of 5 seconds, then resets back to the starting color and restarts the animation:

Direction

The animation-direction property is used to control the direction in which an animation is played; forwards, backwards, or alternating:

.example3 {
  height: 40px;
  width: 40px;
  background: red;
  animation-name: Trafficlight;
  animation-duration: 5s;
  animation-iteration-count: infinite;
  animation-direction: forward;
}
@keyframes Trafficlight {
  0% {background: red;}
  33% {background: orange;}
  66% {background: yellow;}
  100% {background: green;}
}

This code takes a simple purple red, changes the background color to orange, then yellow and finally green.

By changing the direction to reverse, the square instead starts out as green, then changes to yellow, then orange, and finally red.

And by changing the direction to alternate, the square first goes through the forwards changes, then proceeds to go through the reverse changes, and repeats.

Timing function

The animation-timing-function property is used to control the speed curve of an animation.

.example4 {
  height: 40px;
  width: 40px;
  background: magenta;
  position: relative;
  animation-name: HorizontalMove;
  animation-duration: 20s;
  animation-iteration-count: infinite;
  animation-timing-function: ease;
}
@keyframes HorizontalMove {
  0% {left: 0vw;}
  100% {left: 90vw;}
}

This code makes a simple magenta square, which moves from the left side of the screen to the right side of the screen. The default setting for the animation-timing-function is ease, which means an animation has a slow start, moves fast in the middle, and slows down again before the end:
