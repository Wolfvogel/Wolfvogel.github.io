Basic JavaScript

Calculations

For adding, subtracting, multiplying and dividing, all the following examples would set variable value to 15;

var value = 5 + 10;
var value = 20 - 5;
var value = 5 * 3;
var value = 30 / 2;

When a variable has a set value, this can be incremented or decremented by 1, in both of these cases resulting in variable value set to 15;

var value = 14;
value++;

var value = 16;
value--;

It can also have a number added to, subtracted from, divided by or multiplied with it, and then have its value set to the result. There are operators which do this in one step, resulting in the following all resulting in variable value set to 15;

var value = 20;
value -= 5;

var value = 11;
value += 4;

var value = 45;
value /= 3;

value = 7.5;
value *= 2;

The remaining number after dividing two numbers can be found with the remainder operator. It can be used to check for even or odd numbers, by getting the remainder of division by 2. It does not work properly with negative numbers. In this case, variable value would be set to 6;

var value = 13 % 7;

Strings

A series of characters enclosed in (single or double) quotes is called a string. A string is immutable, meaning that once created the individual characters can't just be changed. The only way to change a string, is to assign it with a new string.

var string = "Hello";

When you need to use a single or double quote inside your string, it can be escaped using a \ so JavaScript knows it's not the end of the string. It's also possible to use single quotes within a string that starts and ends with double quotes, and vice versa.

var string = 'I\'m a string';
var string = "I'm a string";

Other things can be represented in a string by using the backslash escape, like;

Backslash: \\
New line: \n
Carriage return: \r (Resets to the beginning of a line of text)
Tab: \t
Word boundary: \b (?)
Form feed: \f (Advance downwards to the next 'page', used as section separators or page breaks)

Strings can be concatenated with +, or concatenated onto the end of an existing string with +=. Concatenating strings does not add spaces between the strings, so make sure to add these yourself where necessary. Strings can also be concatenated with variables.

var doubleString = "This is a " + "concatenated string.";

var doubleString = "This is also a ";
doubleString += "concatenated string.";

var holiday = "Halloween";
var longString = "My favourite holiday is " + holiday + ", what is yours?";

The length of a string value can be found with .length;

var catNameLength = 0;
var catName = "Calida";
catNameLength = catName.length;

It is also possible to find a specific character in a string. For this, remember that computers start counting at 0; the first character in a string is the zeroeth character for your script.

var firstLetter = "";
var catName = "Calida";
firstLetter = catName[0];

var thirdLetter = "";
var catName = "Calida";
thirdLetter = catName[2];

var catName = "Calida";
var lastLetter = catName[catName.length - 1];

var catName = "Calida";
var secondToLastLetter = catName[catName.length -2];

Arrays

It's possible to store multiple pieces of information in a variable, using an array. An array declaration starts and ends with square brackets, and have a comma between each entry. Arrays can also hold other arrays, called a multi-dimensional array;

var pets = ["Cat", 2], "Dog", "Rat", 3]];

The data inside an array can be accessed using indexes, which are written in the same bracket notation. Like strings, they start counting at 0.

var shoppingList = ["Milk", "Bread", "Eggs"];
shoppingList[1]; //equals "Bread"

var multiArray = [[1, 2, 3], [[4, 5], 6], 7, 8, 9];
multiArray[3]; //equals 8
multiArray[0]; //equals [1, 2, 3]
multiArray[1][0][1]; //equals 5

Unlike strings, arrays are mutable and can be changed;

var shoppingList = ["Milk", "Bread", "Eggs"];
shoppingList[2] = "Jam"; //equals ["Milk", "Bread", "Jam"]

Data can be added to the end of an array via the push() function;

var array = [1, 2, 3];
array.push(4); //array equals [1, 2, 3, 4]

The pop() function can be used to remove the last value of an array, and can be used to store that value in a variable.

var array = [1, 2, 3];
var lastValue = array.pop(); //array equals [1, 2]; lastValue equals 3

The shift() function works similar to pop(), except that it removes the first value instead of the last one.

var array = [1, 2, 3];
var firstValue = array.shift(); //array equals [2, 3]; firstValue equals 1

The unshift() function can then be used to add a value to the beginning of an array;

var array = [1, 2, 3];
array.unshift(0); //equals [0, 1, 2, 3]

Switch Statements

A switch statement can test a value against many predefined possible cases, using strict comparison (= = =). It executes statements from the first matched case value until it encounters a break. Cases can be chained, when several cases would produce the same return (in the case below, for values 3 and 4). A default statement can be added, which will be executed if no other cases match the input value.

function(value) {
  var answer="";
  switch (val) {
   case 1: return "one"; break;
   case 2: return 2; break;
   case 3: case 4: return "three or four"; break;
   case "unknown": return "that is not a number"; break;
   default: return "other"; break;
  }
  return answer;
}
